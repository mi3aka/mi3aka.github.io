/var/www/html/.backup

/var/www/html/site/wordpress/config.php

但用户名和密码均不正确…

/home/jangow01/user.txt

cmd5反查得知其为空字符的md5

反弹shell,发现存在端口限制,试了试80,443等常用端口,发现443端口不受影响

尝试切换用户jangow01,返回su: must be run from a terminal,需要将反弹回来的shell转换为交互式的shell

  1. python3 -c 'import pty;pty.spawn("/bin/bash")'

  2. python3 -c "__import__('subprocess').call(['/bin/bash'])"

建议使用第一种方法,成功利用密码abygurl69切换到用户jangow01
